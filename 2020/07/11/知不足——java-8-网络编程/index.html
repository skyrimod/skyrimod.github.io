注意事项

  1. 客户端必须指明服务端的ip和端口号；
  2. 服务端的端口号必须与客户端指定的一样；
  3. ip和端口号信息存储在socket对象中；
  4. 网络通信的流是通过socket对象的静态工厂方法创建的；
  5. 根据传输文件的不同，需要选择不同的流；
  6. 客户端发送完数据后，应该使用socket.shutdownOutput()方法关闭传输，防止服务端一直等到获取数据（阻塞）；
  7. 读写操作与IO流读写操作一样，这里应该理清思路；
  8. 关闭流之前应该对流进行判空。
